Игровые параметры

Глобальные настройки, применяются «на горячую» — для новых спавнов и начислений сразу, без перезапуска. Сохраняются в БД.

Статы аватаров

Переопределение базовых статов героев. Применяется при следующем входе в бой. Пустая/совпадающая ячейка = авторское значение. Изменённые строки подсвечены.

СкАтк = атак/сек · Дальн/Бр = усл. ед.

Статы мобов

Переопределение базовых статов мобов (уровень 1; в бою домножается на уровень места). Применяется к новым спавнам. Боссы помечены ★.
